# CommodityNode SNS Strategy — Influencer Playbook

## 철학
- 매일 1개 플랫폼에 1개 게시물 (퀄리티 > 양)
- 시사 이슈 → 원자재 연결 (Why it matters NOW)
- 데이터로 말하되, 강한 주장으로 마무리
- 커뮤니티와 대화 — 포스팅 후 댓글 적극 참여

## Twitter 스레드 게시 방법 (중요!)
1. 메인 트윗 올리기 (훅 = 첫 1~2줄이 전부)
2. 자기 트윗에 **댓글**로 2번째 트윗 달기
3. 계속 자기 댓글에 댓글 달아서 스레드 이어가기
4. 마지막에 "Full analysis: commoditynode.com" 링크
5. 메인 트윗만 리트윗/공유됨 → 훅이 핵심

## Threads 게시 방법
- 트위터와 동일한 스레드 방식
- 첫 포스트 올리고 → 자기 포스트에 댓글 달기
- 이모지 적극 활용 (MZ 감성)

## Reddit 전략
- 타이틀이 80% — 클릭베이트이되 실망 없어야
- 본문은 진짜 분석 (스팸 아님)
- 포스팅 후 1~2시간 내 댓글 적극 답변 (알고리즘 부스트)
- 첫 1시간이 중요 — 업보트 쌓이면 프론트페이지

## 콘텐츠 품질 체크리스트
발행 전 반드시 확인:
- [ ] 최소 3개 이상 실제 수치/통계 포함?
- [ ] 시사 이슈와 원자재 인과관계 명확?
- [ ] 강한 주장 있음? ("most mispriced asset" 수준)
- [ ] 논쟁 유발? (반박하고 싶게 만드는가)
- [ ] 마지막에 질문으로 마무리?
- [ ] commoditynode.com 링크 자연스럽게 삽입?
