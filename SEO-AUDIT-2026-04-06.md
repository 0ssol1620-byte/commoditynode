## 최우선 긴급 수정 (즉시)

### CRIT-1: vercel.json 보안 헤더 완전 부재
**파일:** `vercel.json`  
X-Frame-Options, CSP, HSTS, X-Content-Type-Options, Referrer-Policy, Permissions-Policy — 6개 헤더 모두 없음. 클릭재킹 취약, Google 신뢰 신호 손상.

```json
"headers": [
  {
    "source": "/(.*)",
    "headers": [
      { "key": "X-Frame-Options", "value": "SAMEORIGIN" },
      { "key": "X-Content-Type-Options", "value": "nosniff" },
      { "key": "Referrer-Policy", "value": "strict-origin-when-cross-origin" },
      { "key": "Permissions-Policy", "value": "camera=(), microphone=(), geolocation=()" },
      { "key": "Strict-Transport-Security", "value": "max-age=63072000; includeSubDomains; preload" }
    ]
  }
]
```

### CRIT-2: 페이지 로더가 LCP를 완전 차단
**파일:** `_layouts/default.html` lines 92-108  
전체 화면 오버레이(position:fixed, z-index:9999)가 모든 페이지에서 동기 렌더링. JS 지연 시 LCP 요소가 무한 숨김. `no_loader: true`가 일반 포스트/커모디티에 없음.

**즉시 수정:**
```javascript
// default.html 로더 JS에 추가
setTimeout(function(){ 
  var l = document.getElementById('page-loader');
  if(l) { l.style.opacity='0'; setTimeout(function(){l.style.display='none';},400); }
}, 1500);
```

### CRIT-3: 커피 포스트 중복 (canonical 없음)

```liquid
<div class="related-companies">
  <h3>Related Companies</h3>
  {% for co in page.companies %}
    <a href="/companies/{{ co | downcase }}/">{{ co }}</a>
  {% endfor %}
</div>
{% endif %}
```
