# About CommodityNode

**CommodityNode** maps the hidden connections between raw material prices and the industries they impact.

When oil prices surge, which sectors win and which lose? When copper demand spikes, what does it signal about the global economy? We answer these questions with data-driven analysis and interactive visualizations.

## What We Do

- **Track** major commodity price movements in real-time
- **Map** the ripple effects through interconnected industries
- **Visualize** complex supply chain relationships as interactive node graphs
- **Identify** investment opportunities created by commodity shifts

## Our Approach

We combine quantitative analysis with intuitive visualizations. Each commodity is a node in a vast economic network — when one moves, others follow in predictable patterns. Understanding these patterns gives investors an edge.
